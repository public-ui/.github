# Welcome to KoliBri

**KoliBri** stands for **Component Library for Accessibility**.

## Vision

We make **HTML accessible** by providing **reusable web components** that ensure **usability** and **accessibility**.

## Mission

The [HTML Standard](https://www.w3.org/html) is intentionally open to remain durable and robust. As a result, HTML structures are often not accessible, semantic or valid by default.

KoliBri builds directly on the [W3C Web Standards](https://www.w3.org/standards/webdesign/) and is framework-agnostic. It serves as a generic reference implementation of the [WCAG](https://www.w3.org/WAI/standards-guidelines/wcag/) and [BITV](https://www.bitvtest.de/bitv_test.html), offering a multi-theme presentation layer without domain-specific logic or data transport.
This makes KoliBri suitable for both static websites and dynamic web applications across different corporate designs and style guides — ideal for open source.

## Collaboration

KoliBri focuses on **small**, **flexible** and **highly reusable** HTML building blocks (e.g., buttons). We provide accessible, semantic and valid implementations that can be combined into higher-level structures (molecules, organisms, templates).

To improve these atomic components sustainably, we invite everyone to **collaborate**, **share knowledge** and **refine the foundation** together.
